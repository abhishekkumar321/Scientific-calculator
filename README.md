# Scientific-calculator
**SCIENTIFIC CALCULATOR MADE IN PYTHON**<br/>
Buttons 0 to 9:To enter numbers<br/>
CE: To clear a single element<br/>
AC: To clear the entry box<br/>
(): For brackets<br/>
'*' : For multiplication<br/>
'+' : For addition<br/>
'-' : For substration<br/>
'/' : For division<br/>
'.' : For decimal<br/>
'=' : For equal to(displays result)<br/>
'sin': For sine<br/>
'tan': For tan<br/>
'cos': For cosine<br/>
'asin': For sine inverse<br/>
'atan': For tan inverse<br/>
'acos': For cosine inverse<br/>
'log' : For logarthmic calculations(base:10)<br/>
'ln' : For logarthmic calculations(base:e)<br/>
'x2' : For squaring a number<br/>
'^' : For x to the power y(x^y)<br/>
'exp': For x to the power of e(e^x)<br/>
'√': For square root<br/>
'deg' : To take input as degree.eg. sin(deg(90)), it will first convert 90 to degrees then apply sine operation.<br/> 
'cDeg': Convert a number to degrees<br/>
'π' :For pi<br/>
